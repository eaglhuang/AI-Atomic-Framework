# Route Interpretation Lessons

Use this shard when ATM returned a valid route, but the operator misread what
the status or playbook implied.

Seed capture targets:

- `framework-temp-claim-required` treated like optional advice instead of a
  pre-edit requirement
- `recommendedChannel` misunderstood as a style preference instead of a work
  order
- `playbook required` acknowledged but not actually followed

When the first real lesson lands here, keep the entry short:

- Trigger
- Symptom
- Correct ATM route
- Durable rule
